# Commercial Boundary

`local-ai-review` OSS Core is free and includes local review, local evidence storage, and local learning.

Review Immune Pro is a commercial extension based on signed, encrypted, annual domain-specific Immune Packs.

## OSS Core

- local LLM PR review;
- diff-only GitHub workflow;
- encrypted local Evidence Vault;
- scoring and calibration;
- dashboard;
- export-safe signal generation, disabled by default.

## Pro Packs

- curated domain-specific bug intelligence;
- offline annual pack files;
- signed license files;
- in-memory pack mount;
- advanced evidence and fix strategy context.

## Promise

The OSS Core should never require a Pro license for basic local review.
